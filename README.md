# Equinix RFP — sample IOS XR configurations (P1 & P2)

Illustrative **Cisco IOS XR** configuration snippets for **P1** and **P2** requirements from the Equinix requirements matrix. **Roadmap** items are excluded.

Each markdown file contains a short overview and a `text`-fenced config example. Validate syntax, scale, and feature support for your **Cisco 8000** release (**K100** / **P100**) before use.

Files are named from the main **Equinix feature** name (see the matrix).
